# Chickpea Mini Muffins

## Ingredients

| Serving | Ingredient | Notes |
|-|-|-|
| 1/4 cup | Rolled oats |  |
| 1 can | Chickpeas |  |
| 1/3 cup | Tahini |  |
| 1/3 cup | Maple syrup |  |
| 2 tsp | Vanilla |  |
| 1/2 tsp | Salt |  |
| 1/2 tsp | Baking soda |  |
| 1/2 cup | Chocolate chips | Semi-sweet |

## Instructions

1. Preheat oven to 350 degrees Farenheit.
1. With the exception of the chocolate chips, blend all ingredients in a blender or food processor.
1. Stir in chocolate chips.
1. Add to an oiled mini muffin tray.
1. Bake for 18-20 minutes.
